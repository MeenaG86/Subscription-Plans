# Subscription-Plans
Subscription Page using Flexbox

A simple and responsive subscription / pricing layout built using HTML and CSS Flexbox.
This project demonstrates how Flexbox can be used to align subscription cards neatly and keep the layout centered across different screen sizes.

✨ Features

Fully responsive subscription layout

Flexbox-based card alignment

Centered heading and pricing cards

Clean and modern UI

Easy to customize plans and styles

🛠️ Technologies Used

HTML5

CSS3

Flexbox

📐 Layout Description

The main container uses display: flex

Subscription cards are aligned using:

justify-content: center

Cards are evenly spaced using gap

Each plan card contains:

Plan title

Price

justify-content
